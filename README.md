# Text Image Processing Tool

Инструмент для обработки изображений с текстом. Позволяет загружать изображения и отмечать на них точки разных цветов.

## Текущий статус проекта

Первый коммит: Создание базового интерфейса для загрузки изображений и отметки точек.

### Функциональность main.py

- Загрузка изображения через диалоговое окно
- Отображение загруженного изображения с автоматическим масштабированием
- Возможность отмечать точки четырех разных цветов:
  - Красный
  - Синий
  - Зеленый
  - Желтый
- Переключение между цветами точек через радиокнопки
- Отображение координат отмеченных точек для каждого цвета
- Возможность очистки всех точек
- Автоматическое удаление точек при загрузке нового изображения

## Установка

1. Создайте виртуальное окружение:
```bash
python -m venv venv
```

2. Активируйте виртуальное окружение:
- Windows:
```bash
venv\Scripts\activate
```
- Linux/Mac:
```bash
source venv/bin/activate
```

3. Установите зависимости:
```bash
pip install -r requirements.txt
```

## Запуск

```bash
python web/main.py
```

## Планируемая функциональность

- Выделение области текста с помощью граничных точек
- Коррекция яркости изображения
- Визуализация выделенной области
